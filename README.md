# keep

> Desktop app for [Google Keep][google-keep] packaged with [Electron][electron]

## Install

```sh
git clone https://github.com/andrepolischuk/keep
cd keep
npm install
```

## Run

```sh
npm start
```

## Build

Build the Mac OS X version:

```sh
npm run darwin
```
